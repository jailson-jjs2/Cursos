Entendendo elementos block

Elemento Block

Exemplos: div, section, ul, li, head

  • Ocupam 100% da largura do elemento pai
  • Ocupam a propria linha, sendo posicionada
  • A altura padrao é definida pelo tamanho
  • É possivel altrar as propriedades
  • É possivel alterar todas as propriedades
